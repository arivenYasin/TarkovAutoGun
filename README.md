# Tarkov Auto GUN

一个用于《逃离塔科夫》的自动配枪工具，  
根据你当前的任务完成情况，商人等级，角色等级等信息，
自动搜索指定的枪在各价位下的最优配装，帮助你在后座、人机和价格之间权衡

---

## 这是什么？

这是一个**逃离塔科夫自动配枪工具**。
设置好你当前等级等信息
输入想要搜索的枪械名称
自动返回你当前可购买的、各个价位所能拿到的最优配枪结果

---

## 如何使用/在哪里下载？

在github当前页面右侧的release页面下载TarkovAutoGun.zip
解压后阅读其中的README.txt文件并按照其内指示操作
