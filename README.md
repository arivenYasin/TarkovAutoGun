# Tarkov Auto GUN

一个用于《逃离塔科夫》的自动配枪工具，  
在满足商人等级、任务解锁和可购买条件的前提下，  
自动搜索并对比不同配枪方案，帮助你在后座、人机和价格之间做权衡。

---

## 这是什么？

这是一个**面向普通玩家的自动配枪工具**。

你只需要输入目标枪械名称和当前角色条件，程序会自动：
- 枚举可行配件组合
- 过滤不满足商人 / 任务条件的方案
- 对不同方案进行对比和可视化展示

无需手动一件一件试配件。

---

## 能做什么？

- 自动搜索可行的配枪方案  
- 在后座 / 人机 / 价格之间进行权衡  
- 支持商人等级、任务完成情况配置  
- 可视化展示不同方案的对比结果  
- 支持已有枪械配置的“指定配装搜索”

---

## 如何使用？

本项目的完整功能以 **Windows 可执行程序** 的形式发布。

- 无需 Python 环境
- 下载解压后即可运行

详细使用说明请参考程序目录内的 `readme.txt`。
